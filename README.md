# 🤖 AI Resume Screener — Powered by Claude API

An intelligent resume screening tool that analyzes candidate resumes 
against job descriptions using the Anthropic Claude API.

## 🚀 What It Does
- Upload multiple PDF resumes at once
- Paste any job description
- Get instant AI analysis for each resume:
  - Match percentage (0–100%)
  - HIRE / MAYBE / NO HIRE verdict
  - Matched skills vs missing skills
  - 2-3 sentence candidate summary
- Clean summary table comparing all candidates

## 🛠️ Tech Stack
- HTML, CSS, JavaScript (Frontend)
- PDF.js (PDF text extraction)
- Anthropic Claude API (AI analysis)
- Live Server (local deployment)

## 💡 Why I Built This
As a fresher job seeker myself, I wanted to understand what makes 
a resume stand out. So I built the tool that screens resumes — 
and then ran my own resume through it.

## ⚙️ How to Run
1. Clone this repo
2. Open in VS Code
3. Add your Anthropic API key in the input field
4. Run with Live Server
5. Upload PDFs and paste a job description
6. Click Screen Resumes!
